# Data Source Truthfulness

## Principle

資料不夠新，就不給強推薦；資料來源只作為說明，不作為降等理由。若 Yahoo 是目前交易狀態下最新有效資料，就正常採用。

## Source Policy

| 情境 | 採用資料 | 推薦狀態 |
|---|---|---|
| 官方資料日期 >= Yahoo 日期 | 官方盤後資料確認最新價 | 可作為操作參考 |
| Yahoo 日期 > 官方資料日期 | Yahoo 較新資料 | 採用最新有效資料，不因來源降等 |
| 官方資料無日期 | 保留 Yahoo | 不讓無日期官方資料覆蓋 |
| 資料早於預期最新交易日 | 不給 A 級買進 | 僅觀察 |
| fallback | 不給買進 | 僅觀察 |

## Expected Latest Date

- 盤前：前一交易日。
- 盤中 / 收盤後整理 / 盤後：今日。
- 非交易日：前一交易日。

未來可加入 TWSE 官方交易日曆，處理國定假日與補班日。
