# Data Source Truthfulness

## Principle

資料不夠新，就不給強推薦；來源不一致，就明確標示；官方資料未更新時，可以採用較新的 Yahoo 資料，但必須讓使用者知道。

## Source Policy

| 情境 | 採用資料 | 推薦狀態 |
|---|---|---|
| 官方資料日期 >= Yahoo 日期 | 官方盤後資料確認最新價 | 可作為操作參考 |
| Yahoo 日期 > 官方資料日期 | Yahoo 較新資料 | 可參考，但信心略降 |
| 官方資料無日期 | 保留 Yahoo | 不讓無日期官方資料覆蓋 |
| 資料早於預期最新交易日 | 不給 A 級買進 | 僅觀察 |
| fallback | 不給買進 | 僅觀察 |

## Expected Latest Date

- 盤前：前一交易日。
- 盤中 / 收盤後整理 / 盤後：今日。
- 非交易日：前一交易日。

未來可加入 TWSE 官方交易日曆，處理國定假日與補班日。
