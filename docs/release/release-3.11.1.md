# Release v3.11.2

## Title

Version Integrity + Report Sync

## Purpose

v3.11.0 改善了 UX，但老師建議仍偏系統化。本版將建議語句重新拉回「可執行、具體價位、少廢話」的方向。

## Changes

- 所有操作建議都必須帶有具體價格或區間。
- 籌碼未取得時，僅簡短標示，不用模板補字。
- 恢復並強化 0 軸轉強雷達。
- MACD 卡片加入小型 MACD 圖與具體轉強條件。

## Acceptance

- 今日可買、持股、MACD 觀察都不應出現泛用的「等支撐」而沒有價格。
- 沒有籌碼資料時，只顯示未取得。
- 0 軸轉強雷達仍可使用，且有 DIF / DEA / Histogram 圖表。
