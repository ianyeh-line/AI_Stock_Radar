# Release 3.5.4 — Teacher Portfolio Language Hotfix

## Fixed

- Removed wording that implied Yahoo newer data lowers confidence when it is the freshest valid data.
- Removed redundant data-trust slogan from portfolio coach recommendations.
- Rewrote portfolio coach summary and per-position advice to be closer to the 股市老師模式: concrete scenarios, support/breakout/stop levels, and discipline-first guidance.
- Added Radar score display in portfolio rows so score-based comments have visible context.

## Product Rule

Fresh valid data is sufficient. Recommendations are only downgraded when data is stale, fallback, missing, or technically insufficient.
