# Release v3.9.0 - Decision Quality Gate

## Fixed

- 修正老師建議沒有依照最新股價相對於拉回區、突破價與今日漲停上限動態改寫的問題。
- 移除「可列入今日可操作名單，但不是無條件追價」等制式文字。
- 今日可買與持股總教練改用更嚴格的價格區間判斷，現價已明顯高於拉回區時不再寫可分批買進。
- 修正 MACD 觀察頁因缺少 `_macd_chart_series` 導致 NameError 的問題。
- 強勢股雷達新增可追條件說明：漲幅、量能、RSI、短期新高與資料可操作性需同時成立。
- 移除無實際法人資料時的籌碼面模板句。

## Product Rule

股市老師建議必須先看現價，再判斷買點是否仍可執行；不能用已經失效或今日不可到達的價格條件給使用者操作建議。
