# Release v3.8.1 - Market Strength Connector Fix

## 目標

修正 v3.8.0 強勢股雷達「宣稱全市場掃描，但實際未取得 / 未解析資料」的問題。

## 本版改進

- 新增 TWSE / TPEx / Yahoo endpoint-level 診斷。
- TWSE / TPEx 官方快照解析失敗時，不再假裝全市場掃描成功。
- 官方快照不足時，補用官方股票主檔 + Yahoo Quote 做較廣市場掃描，並明確標示。
- 強勢股頁面新增「資料抓取診斷」，顯示每個 endpoint 的原始筆數、解析筆數與錯誤。
- 強勢股分類仍分為可追強勢、今日強勢、漲停/接近漲停、已漲不追、明日接力。

## 驗收重點

1. 強勢股雷達不應只顯示空白且沒有原因。
2. 若官方資料抓不到，頁面要明確顯示哪個 endpoint 失敗。
3. 若官方資料不足但 Yahoo Quote 成功，強勢股雷達應可產生候選，且標示為補充資料。
