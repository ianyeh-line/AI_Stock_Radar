# Release v3.7.0 - Strong Momentum Radar

## Goal

讓 AI Stock Radar 同時回答：

1. 今天哪些股票可以按照波段買點操作？
2. 今天市場真正強勢在哪裡？
3. 哪些股票強但不能追？
4. 哪些股票適合作為明日接力觀察？

## Key Changes

- 新增 Strong Momentum Radar engine。
- 新增 Dashboard「強勢股雷達」功能頁。
- 新增每日報告強勢股區塊。
- 新增 CLI 強勢股摘要。
- 新增 AI 推薦與今日強勢股落差分析。

## Product Principle

強勢不等於可買。

股市老師應該同時說清楚：

- 股票為什麼強？
- 現在還能不能買？
- 如果不能追，明天看什麼條件？
