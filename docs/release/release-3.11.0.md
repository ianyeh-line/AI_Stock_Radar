# Release v3.11.0

## Name

Responsive Decision UX + Chip Data Foundation

## Goals

1. 讓手機與桌面都能快速閱讀與操作。
2. 把與決策無關的系統資訊預設收合。
3. 把產品主頁從功能展示收斂成決策工具。
4. 建立籌碼資料基礎版：有官方三大法人資料就納入，沒有就明確說沒有，不用模板假裝。

## Scope

- App UX restructuring.
- Responsive CSS and card layout.
- TWSE T86 chip data connector foundation.
- Chip status in decision cards.
- Documentation updates.

## Out of scope

- Full chip trend engine.
- Multi-day institutional consecutive buy/sell.
- Full backtesting.
