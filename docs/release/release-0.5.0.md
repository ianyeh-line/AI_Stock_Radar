# Release v0.5.0

## Summary

This release introduces Decision Cards as the core product unit of AI Stock Radar.

## Why

Previous versions produced a ranking but did not explain why each stock received a decision. v0.5.0 makes each stock decision explainable and reviewable.

## Run

```bash
PYTHONPATH=src python3 -m radar.cli run
```

## Output

```text
output/daily_report.md
```
