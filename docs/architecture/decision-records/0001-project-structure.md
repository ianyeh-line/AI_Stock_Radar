# ADR 0001: Project Structure

## Status

Accepted

## Context

AI Stock Radar must be maintained as a long-term product rather than a prototype.

## Decision

Use a modular repository structure with separate areas for documentation, source code, configuration, tests, scripts, and data.

## Consequences

- Clear module ownership
- Easier onboarding
- Documentation and code evolve together
- Future automation and deployment can be added without restructuring the project
