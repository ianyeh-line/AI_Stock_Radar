# PM Precision Action Architecture

v1.4.0 將 Decision Card 延伸為 PM Action Card。

## Inputs

- Daily price bars
- MA5 / MA10 / MA20 / MA60 / MA120
- MACD / RSI
- Volume ratio
- News impact chain
- Stock universe metadata
- Investor profile

## Output

Each card now includes:

- Breakout price
- Pullback zone
- Reduce price
- Stop / invalidation price
- Volume ratio explanation
- PM-style swing action language

## Limitation

This version uses daily bars and RSS news. It is not an intraday real-time trading system.
