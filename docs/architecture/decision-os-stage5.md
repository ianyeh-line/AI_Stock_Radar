# Decision OS Stage 5 Architecture

## Pipeline

```text
RSS News / Fallback News
        ↓
Chinese Investment Summary
        ↓
Knowledge Mapping
        ↓
Technical Radar
        ↓
MACD Reversal Screener
        ↓
Swing Portfolio Manager
        ↓
Decision Cards
        ↓
Dashboard / Markdown Report
```

## Stage 5 Principle

AI Stock Radar 不只評分，而是以波段投資經理人的方式給出：

- 操作方向
- 進場條件
- 加碼條件
- 停損條件
- 停利條件
- 持有週期
- 風險提醒
