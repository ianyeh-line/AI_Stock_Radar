# Sprint 3 - Explainable Decision

## Goal

Make AI Stock Radar useful as a decision product, not only a report generator.

## Delivered

- Explainable Decision Cards
- Evidence chain by stock
- Radar Score + Decision + Confidence
- Action recommendation per stock
- Risk alert

## Acceptance Criteria

- User can run one command.
- Product generates `output/daily_report.md`.
- Each Top 5 stock has visible evidence and action.
