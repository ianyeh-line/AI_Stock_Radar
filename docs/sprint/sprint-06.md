# Sprint 6 - Investment Manager Layer

## Objective

把 AI Stock Radar 的產品輸出從排行表升級為專業投資經理人早會決策。

## Scope

- PM Brief
- Score Breakdown
- Capital Allocation
- Top Actions
- Avoid Actions
- Invalidation Criteria
- Data Quality Check

## Definition of Done

- CLI 可執行
- Dashboard 可開啟
- 每檔股票具備可解釋分數拆解
- 使用者可在 3 分鐘內理解今天的主策略
