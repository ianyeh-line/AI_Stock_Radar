# Sprint 21 - User Account + Cloud Portfolio

## Goal

Make AI Stock Radar shareable without forcing friends to re-enter their holdings
each time they open the website.

## Delivered

- Auth-aware Dashboard.
- Cloud user store integration.
- Supabase schema.
- Streamlit Secrets example.
- Cloud portfolio/watchlist persistence.

## Validation

- CLI remains executable.
- Dashboard still works without secrets in Guest Mode.
- User-specific storage activates only when Google Login and Supabase are configured.
