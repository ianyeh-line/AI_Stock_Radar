# Data Freshness Rule v1

## Core Principle

只要資料是依照目前交易狀態可取得的最新資料，就不因資料來源或資料型態而降等。

## Valid Fresh Data

- 交易日盤前：前一交易日收盤資料有效。
- 交易日盤中：當下盤中資料有效，但標示為盤中資料。
- 交易日盤後：今日最新可得資料有效，不限官方或 Yahoo。
- 非交易日：最近交易日資料有效。

## Downgrade Only When

- 資料早於目前交易狀態應有的最新資料。
- 資料是 fallback / 模擬資料。
- 資料抓取失敗或日期無法判斷。
- 技術指標樣本不足。

## Explicitly Removed

不同資料來源價格差異過大，不再單獨作為降等條件；系統以最新資料日期與可用性作為主要判斷。
