# Release v3.6.1 - Teacher Narrative Fix

## Purpose

修正 v3.6.0 在「今日可買」與「股市老師分析」仍過度模板化的問題，讓今日可買、持股總教練與每日報告共用一致的股市老師敘事邏輯。

## Fixed

- 今日可買不再只顯示簡短理由，改為顯示股市老師完整分析。
- 移除推薦理由中的資料來源降等語句，例如 Yahoo 較新、官方尚未同步、信心略降。
- 今日可買、持股總教練共用同一套 Teacher Narrative Engine。
- 股市老師分析新增更清楚的技術面、籌碼/法人面、產業/消息面、支撐壓力與 A/B/C 劇本。
- 持股總教練不再以資料可信度口號干擾操作建議。
- MACD 觀察卡移除資料來源說明，保留 MACD 小圖與老師判斷。

## Verification

- CLI 可執行。
- Tests passed: 23.
