# AI Stock Radar

AI Stock Radar is a decision-support product designed to help Taiwan stock investors complete a daily market decision within 3 minutes.

The product is not a technical analysis demo site. It is a maintainable decision engine that combines market news, macro signals, technical indicators, institutional flow, and explainable scoring into a daily action report.

## Product Goal

Help users answer every morning:

1. What happened in global markets?
2. What matters to Taiwan stocks today?
3. Which stocks are worth watching?
4. What action should I take: Buy, Watch, Hold, Reduce, or Avoid?

## Core Workflow

```text
Market Data
  -> News Intelligence
  -> Market Impact Analysis
  -> Technical Confirmation
  -> Radar Score
  -> Daily Action Report
```
